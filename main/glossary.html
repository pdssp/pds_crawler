ODE web service¶

The Orbital Data Explorer (ODE) is a web-based search https://oderest.rsl.wustl.edu/ developed at NASA’s Planetary Data System’s (PDS) Geosciences Node. Through ODE, users can search, browse, and download a wide range of PDS Mars, Moon, Mercury, and Venus data.

Planetary Data Systems

PDS Data Repositories¶

combines the ODE web service and the PDS dataset browser

PDS dataset browser¶

The Data Set Browser allows the user to view data set contents that are currently cataloged in the ODE system. It exists several dataset browser:

  • Mercury https://ode.rsl.wustl.edu/mercury/datasets

  • Venus https://ode.rsl.wustl.edu/venus/datasets

  • Moon https://ode.rsl.wustl.edu/moon/datasets

  • Mars https://ode.rsl.wustl.edu/mars/datasets
