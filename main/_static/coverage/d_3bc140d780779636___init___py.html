6"""The system architecture consists of several components that work together to crawl 

7planetary data. The main component can be grouped in two layers: 

8 

9* persistence layer that includes three storage systems: STAC_Storage, PDS_Storage, and HDF5_Storage, which store data in the File System. 

10* busisness layer contains the Extractor and Transformer components. 

11 

12The Extractor component extracts data from the Planetary Data System, which comprises 

13a web service and a website and sends the retrieved data to both PDS and HDF5 storage. 

14The Transformer component then transforms the data into the STAC format and stores it 

15in the STAC_Storage. 

16 

17The Models component, used by layers, includes four groups of models: 

18 

19* PDS3 objects - catalogs, 

20* ODE WS - collections, 

21* ODE WS - records, 

22* and STAC. 

23

103In summary, the architecture consists of three storage systems, an extractor that 

104retrieves data from various sources, a transformer that converts the data into the 

105STAC format, and a web service and website component that provides access to catalogs. 

106Finally, the models component includes four models that represent the different types 

107of data that the system manages
